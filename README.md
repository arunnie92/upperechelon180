# Upper_Echelon_180

Scripts for Phantom

### To run all scripts:

```
./main.sh
```

### To run Go script to create profiles:

```
go run main.go
```

### To run Node script to read excel sheet:

```
node index.js
```

### Delete all profiles:

```
rm -f ~/Library/ApplicationSupport/Phantom/ProfileManager.json
```
